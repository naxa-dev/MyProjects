## Deep_fake (Roop 기반)

### 개요
Roop 기반 얼굴 교체(face swap) 기능을 Streamlit 웹앱으로 구성한 개인 프로젝트입니다.  
로컬에서 업로드/실행 가능한 UI를 제공하며, GPU 없이 CPU 환경에서도 동작하도록 구성했습니다.

- **대상:** 정면에 가까운 짧은 영상/이미지
- **제한:** 처리 시간/성능을 고려해 영상 길이 **약 5~15초**

### Tech Stack
- **Language:** Python
- **UI:** Streamlit
- **Base:** roop (v1.3.2)
- **Runtime:** CPU-only 지원

### 주요 기능
- 소스 이미지/타겟 영상 업로드 UI
- 옵션/기본 설정 선택 및 실행
- 결과 영상 미리보기/다운로드
- 입력 형식/길이 제한 및 예외 처리

### 제한 사항
- CPU 환경에서는 영상 길이/해상도가 커질수록 처리 시간이 크게 증가합니다.
- 입력 데이터(각도/조명/해상도)에 따라 결과 품질 차이가 발생합니다.
- 오남용/개인정보 이슈가 있을 수 있어 테스트 범위를 제한하는 것을 권장합니다.
