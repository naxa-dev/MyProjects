## Auto_keyword (키워드 자동 추출 도구)

### 개요
구글 자동완성, 네이버 연관검색어, 네이버 검색광고에서 제공하는 키워드 신호를 수집·조합하여  
**콘텐츠 제작, 블로그 SEO, 마케팅 리서치에 활용 가능한 키워드 리스트를 자동으로 생성**하는  프로젝트입니다.

---

### 흐름도

> 사용자/스케줄러 → 저장된 키워드 입력 →  
> (구글 자동완성 + 네이버 연관검색어 + 네이버 광고 키워드 수집) →  
> 키워드 병합/정제 → 결과 저장

---

### Tech Stack
- **Language:** Python
- **Crawling/Automation:** Selenium, requests
- **Parsing:** BeautifulSoup
- **External APIs:**  
  - Google Search (자동완성 기반 수집)  
  - Naver 연관검색어  
  - Naver Search Ad API (키워드 플래너)
- **Output:** TXT / CSV / JSON
- **Runtime:** Local (Headless Chrome 지원)

---

### 주요 기능 (Functional)

#### 1) 구글 자동완성 키워드 수집
- Selenium 기반 자동완성 키워드 수집
- 실시간 검색 흐름을 반영한 파생 키워드 확보
- Headless 모드 지원으로 자동화/서버 환경 실행 가능

#### 2) 네이버 연관검색어 수집
- 네이버 검색 결과 기반 연관 키워드 수집
- 국내 사용자 검색 패턴에 최적화된 키워드 확보
- 단일 키워드 → 다수 확장 키워드 생성

#### 3) 네이버 광고 키워드 연동
- 네이버 검색광고 API를 통한 키워드 조회
- 마케팅/상업성 높은 키워드 확보
- (옵션) 검색량·경쟁도 기반 필터링 확장 가능

#### 4) 키워드 병합 및 정제
- 플랫폼별 수집 키워드 병합
- 중복 키워드 제거
- 공백/특수문자/불필요 표현 정리
- 길이/형태 기준 키워드 필터링

#### 5) 결과 저장 및 활용
- TXT / CSV / JSON 형태로 결과 저장
- 블로그 작성, SEO 분석, 마케팅 키워드 리서치에 바로 활용 가능
